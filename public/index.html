About

Hashpass is not a password manager; rather, it is a simple but secure password "calculator".

Unlike traditional password managers, which save your passwords in an insecure database, Hashpass uses cutting-edge cryptographic hashing algorithms to produce random characters procedurally.

Hashpass may utilize the master password, combined with a website name, to "calculate" the same password for that site every time as long as you remember a strong and unique master password.

Each master password and site name combination leads to a secure 16-character password that even the most powerful supercomputers cannot decipher.

Our approach employs Argon2 hashes, which are hundreds of times more secure than standard algorithms such as MD5 or SHA256.

Furthermore, by using our proprietary Systematic Multi-Index Replacement Algorithm (SMIRA), we ensure that all created passwords include lowercase, uppercase, numerical, and special characters, which most other password generators do not.
